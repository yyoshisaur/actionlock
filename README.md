# actionlock

アクション後の硬直をカウントダウンするアドオン。目安程度。

- カウントダウンの設定

        //alock spell/ws/ja/range time
    
    例) ウェポンスキル後のカウントダウンを10秒に設定

        //alock ws 10

- カウントダウンの表示

        //alock on/off
        //alock enable/disable

- プラグインのTimersにカウントダウンを表示

        //alock timers on/off
        //alock timers enable/disable

- 設定をdata/setting.xmlに保存

        //alock save

- ヘルプを表示

        //alock
